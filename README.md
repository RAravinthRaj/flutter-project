# Thought Dump - AI Organizer

Thought Dump is a production-shaped Flutter mobile app backed by Firebase. Users can write unstructured thoughts, and the system organizes them into tasks, ideas, and worries.

## What is included

- Flutter mobile app with Riverpod and feature-based structure
- Firebase Authentication with anonymous sessions and Google sign-in flow
- Firestore-backed real-time thought sync
- Cloud Functions scaffold for AI classification
- Offline-friendly Firestore usage
- Search, filtering, dashboard stats, dark mode, and voice-to-text input
- Firestore security rules and Firebase project scaffolding files

## Firebase setup

1. Install the Firebase CLI:

```bash
npm install -g firebase-tools
```

2. Log in:

```bash
firebase login
```

3. Install FlutterFire CLI:

```bash
dart pub global activate flutterfire_cli
```

4. Configure Firebase from the project root:

```bash
flutterfire configure
```

5. Replace the placeholder [`lib/firebase_options.dart`](/Users/aravinthraj/Developer/flutter-project/lib/firebase_options.dart) with the generated file from `flutterfire configure`.

6. In Firebase Console, enable:

- Authentication
- Anonymous provider
- Google provider
- Cloud Firestore
- Cloud Functions

## Google sign-in notes

- Android: ensure the package name matches your Firebase Android app.
- iOS: add the correct `REVERSED_CLIENT_ID` URL scheme from `GoogleService-Info.plist`.
- After `flutterfire configure`, add the platform files generated by Firebase if they are not already present.

## Cloud Function setup

Install functions dependencies:

```bash
cd functions
npm install
```

Optional AI provider secret:

```bash
firebase functions:secrets:set OPENAI_API_KEY
firebase functions:secrets:set OPENAI_MODEL
```

Deploy functions and Firestore rules:

```bash
firebase deploy --only functions,firestore
```

If no AI secret is configured, the Cloud Function falls back to a built-in heuristic classifier so the app still works.
